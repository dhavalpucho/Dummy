# Dummy Single Page Application

A simple, lightweight single page application (SPA) built with vanilla HTML, CSS, and JavaScript.

## Features

- ✨ **Smooth Navigation** - Navigate between Home, About, and Contact pages without page reloads
- 📱 **Responsive Design** - Works perfectly on all devices (mobile, tablet, desktop)
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations
- 📋 **Contact Form** - Functional contact form with validation
- ⚡ **No Dependencies** - Pure vanilla JavaScript, no frameworks needed
- 🚀 **Fast & Lightweight** - Minimal code, maximum performance

## Project Structure

```
Dummy/
├── index.html       # Main HTML file
├── styles.css       # Styling and responsive design
├── script.js        # JavaScript functionality
├── README.md        # This file
└── .gitignore       # Git ignore file
```

## Files Description

### index.html
- Main entry point of the application
- Contains three pages: Home, About, Contact
- Navigation buttons to switch between pages
- Contact form with input fields and submit button

### styles.css
- Professional gradient design (purple theme)
- Responsive layout that adapts to all screen sizes
- Smooth animations and transitions
- Hover effects and interactive states

### script.js
- Page navigation logic
- Contact form validation
- Email validation using regex
- Form submission handling with success/error messages

## How to Use

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **Use navigation buttons** to switch between pages
4. **Try the contact form** on the Contact page

## Deployment with GitHub Pages

1. Go to your repository settings
2. Navigate to **Settings → Pages**
3. Under "Build and deployment":
   - Select **Deploy from a branch**
   - Choose **main** branch
   - Select **/root** folder
4. Click **Save**

Your site will be live at: `https://dhavalpucho.github.io/Dummy/`
